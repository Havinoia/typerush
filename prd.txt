🚀 TypeRush – Speed Typing Test App
1. 📌 Overview

TypeRush adalah aplikasi web interaktif yang dirancang untuk mengukur kecepatan dan akurasi mengetik pengguna secara real-time. Aplikasi ini menyediakan pengalaman latihan mengetik yang cepat, responsif, dan kompetitif dengan fitur analisis performa.

2. 🎯 Tujuan Produk
Membantu user mengukur kemampuan mengetik (WPM & akurasi)
Menyediakan latihan mengetik yang engaging
Menjadi project portfolio yang menunjukkan skill frontend & backend
(Opsional) Membangun sistem leaderboard kompetitif
3. 👤 Target User
Pelajar / mahasiswa
Programmer / content writer
User umum yang ingin meningkatkan typing skill
